units

%-> percentage(screen resolution)

px ->pixel(1px=1/96 inch)

mm ->millimeter

cm ->centimeter

pt ->point(1point =1/72 inch)

ch ->width of 'o' glyph in the font size

em ->em,rem(1em=16px)(element)(rem - responsive element)

pc->pica(1 pica=12points)

ex->X-height of the element's font

gd -> grid defined by layput -grid

vh & vw ->viewport height and width (current device resoultion)
